# My Dev Env Setup

## TMUX

```bash
brew install tmux
```

```bash
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`
```

```bash
# type this in terminal if tmux is already running
tmux source ~/.tmux.conf
```

Inside tmux, enter <kbd>ctrl - b</kbd> + <kbd>I</kbd>
